# Ducksite Frontend (Rebuilt)

This frontend is rebuilt with Next.js App Router and a minimal UI.

## Setup

1. Copy .env.example to .env.local
2. Install deps: npm install
3. Run dev: npm run dev

## Routes

- /login
- /signup
- /admin-signup
- /dashboard
- /admin/dashboard
- /admin/users
- /admin/classes
- /admin/courses
- /admin/materials
- /admin/challenges
- /admin/progress
